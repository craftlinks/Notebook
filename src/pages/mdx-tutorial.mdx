# Astro + MDX Quick Tutorial

MDX lets you write Markdown sprinkled with JSX/HTML and even interactive components.

## 1. Basics

You can write normal Markdown like **bold text**, lists, and code blocks:

```md
- This
- is
- markdown!
```

## 2. Adding JavaScript expressions

Define a variable and inject it right into the page:

```mdx
export const name = "CraftLife";
```

## 3. Importing and using components

Any supported component (`.astro`, `.jsx`, `.svelte`, `.vue`, etc.) can be imported and used just like in JSX:

```mdx
import Example from '../example.astro';

<Example />
```
